Conectar sua IA ao IntellexIA

O IntellexIA possui um conector MCP que permite ao Claude consultar os dados do seu escritório direto na conversa: painel FAP, contestações, benefícios, base de conhecimento, processos e relatórios em Excel — sempre com o seu login e respeitando suas permissões.

Como conectar no Claude Desktop / claude.ai
  1. Abra Settings → Connectors → Add custom connector;
  2. Cole a URL do conector (abaixo) e confirme;
  3. O navegador abre no IntellexIA — clique em Autorizar e pronto.

No Claude Code (terminal): claude mcp add --transport http intellexia {{ mcp_public_url }}
